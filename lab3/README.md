# Minesweeper game - Сапер

## Запуск

```bash
cd lab3
python -m pip install -r requirements.txt
python main.py
```

## Главное меню

- `начать игру`
- `таблица рекордов`
- `справка`
- `выход`

## Реализованные требования

- Событийно-ориентированное графическое приложение на `pygame`.
- Все настройки вынесены во внешние JSON:
  - `config/game_config.json`
  - `config/network_config.json`
  - `data/records.json`
- Режим с ограничением времени (обратный отсчёт, проигрыш по истечении времени).
- Онлайн-игра на двоих (LAN Host/Join, TCP, JSON-пакеты).
- Таблица рекордов (только одиночный timed-режим).
- Диалог поздравления и ввод имени при новом рекорде #1.
- Звуки и фоновая музыка.
- Анимации:
  - анимация открытия клетки
  - анимация взрыва при проигрыше
  - анимация пульса при победе

## Формула подсчёта очков

```text
score = difficulty_base + revealed_safe * 10 + time_left * 5 - wrong_flags * 15
```

## Онлайн протокол

Поддерживаемые типы сообщений:

- `hello`
- `match_config`
- `state_update`
- `finish_report`
- `match_result`
- `error`
- `disconnect`

Определение победителя:

- Если оба очистили поле: быстреее время, при равенстве победитель - игрок с большим счетом.
- Если очистил только один — он победил.
- Если никто не очистил — победитель по числу очков, при равенстве ничья.

## Управление

- В меню: мышь.
- В игре:
  - ЛКМ: открыть клетку
  - ПКМ: флаг
  - `Esc`: выход в меню
